# WealthWise
Анализатор личных финансов помогает пользователям анализировать выписки с банковских карт, выявляя паттерны трат и предоставляя рекомендации по оптимизации бюджета. Веб-интерфейс позволяет загружать выписки, а AI-аналитик автоматические классифицирует транзакции и прогнозирует расходы.
